4. Using AJAX to Consume a JSON Web Service**

**Pay special attention to this one. The rubric and other parts of the course are misleading and most people think they just need to use AJAX to read the content of a file. That is NOT the requirement. The requirement is for you to call a 3rd party Web Service (aka API). This is the most complex of the topics, so start reading up on it if you don't know much about it. I'll will send out more info about it in a few weeks.
